Conexión BD con PHP y MySQL
Antes de trabajar con MySQL, vamos ver los queries más cómunes:

Para mostrar los registros de:

• SELECT * FROM tabla [Muestra todos los resultados de BD]
• SELECT id,nombre,apellido FROM tabla [Muestra todos los resultados de BD pero solo las columnas id, nombre y apellido]
• SELECT * FROM tabla WHERE id = '1' [Muestra los resultados que tengan la columna con id 1]
• SELECT * FROM tabla WHERE pais = 'Italia' ORDER by id ASC [Muestra los resultados que tengan la columna país con valor
Italia ordenándolo por la columna id en modo ascendente]
• SELECT * FROM tabla ORDER by id DESC LIMIT 10 [Muestra los resultados de BD en orden descendeente por columna id limitándolo solo a 10 resultados]
• SELECT * FROM tabla ORDER by RAND() [Muestra los resultados de BD en un orden aleatorio]

Para agregar registros:

• INSERT into tabla (nombre, apellido, pais, idioma) VALUES (Heidenber, 'Palacio', 'Colombia', 'ES') [Agrega un registro con nombre: Heidenber, apellido: Palacio, país: Colombia, idioma: ES de español]
• INSERT into tabla (id, nombre) VALUES (NULL, '$_POST[nombre]') [Agrega en la base de datos un registro con id NULL, es decir sería automático o vacía dependiendo de la estructura de la columna, y con el nombre tomado de un campo de formulario en PHP]

Para editar registros:

• UPDATE tabla SET nombre = 'Matías', apellido = 'Fernandez', pais = 'USA', idioma = 'EN' WHERE id = '1' [Modifica el registro con ID 1 con los valores nombre: Matías, apellido: Fernández, pais: USA, idioma: EN de english]
• UPDATE tabla SET Idioma = 'EN' WHERE apellido = 'Palacio' AND nombre = 'Heidenber' [Modifica el idioma a EN de english donde el nombre sea Heidenber y el apellido sea Palacio]

Para eliminar registros:

• DELETE FROM tabla WHERE id = '1' [Borramos el registro]
• DELETE FROM tabla WHERE nombre = 'Heidenber' [Borramos todos los registros que tengan el nombre de Heidenber]

Para vaciar una tabla:

• TRUNCATE TABLE tabla [Vacía todos los registros de la tabla]
